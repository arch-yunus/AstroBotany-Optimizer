# Aeroponic Module Design (v1.0)

## Overview
This document serves as the technical specification for the AstroBotany-Optimizer mechanical assembly. The system is designed for high-density microgreen production in microgravity or low-gravity (Lunar/Martian) environments.

## Component Specifications
- **Growth Chamber:** 40x40x60cm pressurized aluminum/polycarbonate composite.
- **Misting Array:** 4x Ultrasonic piezoceramic atomizers (5um - 50um droplet size).
- **Nutrient Distribution:** Closed-loop pressurized manifold with inline UV-C sterilization.
- **Root Support:** Elastic mesh with root-zone separation to prevent anaerobic conditions.

## CAD Files (Conceptual)
Actual STEP/STL files are managed in the `cad_models/` directory for production.
- `AEROPONIC_MODULE_V1.STEP`: Master assembly.
- `NOZZLE_MOUNT_4X.STL`: 3D printable atomizer mounts.
- `ROOT_TRAY_SLIDER.STL`: Removable crop tray mechanism.
